Hoe gebruik ik Actionbound? (bound)

  • Bound aanmaken stap 1

    Na uw account aan te maken zou u deze knop moeten zien op uw scherm. Klik op deze knop om een nieuwe Bound aan te maken.

  • Bound aanmaken stap 2

    Hier kan u een naam voor uw nieuwe bound kiezen. Geef ook een url voor uw bound. Weet dat u deze url later niet meer kan aanpassen.

  • Bound aanmaken stap 3

    Hier kan u verschillende soorten elementen toevoegen.

  • Bound aanmaken stap 4

    Hier kiest u welk element u wilt toevoegen.

  • Bound aanmaken stap 5

    Om andere mensen te laten deelnemen kan u de link delen. U kan de code van de Bound aanpassen als u dit wilt.
